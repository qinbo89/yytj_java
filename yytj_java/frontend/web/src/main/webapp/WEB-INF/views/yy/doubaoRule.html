“一元夺宝”指用户花费1元参与一个商品的夺宝行动,用户可自定义一个号码作为夺宝号，最终抽取一个幸运号作为最终获奖号码。

每件商品在满足开奖条件后，根据夺宝规则抽取出的一个号码。所有夺宝号为该幸运号码的用户都可获得该商品。

参与规则

1. 参与商品夺宝：商品有对应数量的夺宝号码，用户花费1元可以拥有一个号码（用户可自定义号码或随机生成）；

2。获得商品：当该商品满足开奖条件后，系统根据规则抽出1个幸运号码，所有夺宝号为该幸运号码的用户都可获得该商品。用户还可以选择收取与该商品等价的现金， 零购官网将会将金额转账至用户零购官网账户。

3.忘记领奖：如用户中奖后忘记选择领取方式，7天后系统默认将与中奖商品等价的金额转至用户的零购账户。

4.如果您的三级内盟友参与零元夺宝，您也将获得奖励哦。
